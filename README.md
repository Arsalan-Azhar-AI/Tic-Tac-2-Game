
# Tic-Tac-2-Game

**Overview:**
A browser-based **Tic-Tac-Toe game** for two players (local, no AI) built with **HTML**, **CSS**, and **JavaScript**. Players take turns marking X or O on a 3×3 grid. The game detects wins, draws, and provides a reset option.

---

## Features

* **Interactive Game Board:** Clickable 3×3 grid where X and O alternate turns.
* **Turn Tracking:** Prevents overwriting an occupied cell; manages X/O turns.
* **Win Detection:** Checks all rows, columns, and diagonals after every move. Announces winner or draw.
* **Reset Functionality:** "Reset Game" button starts a new round instantly.
* **Responsive Design:** Works on desktop and mobile devices with a clean interface.

---

## Tech Stack

* **Frontend:** HTML5, CSS3
* **Logic:** Vanilla JavaScript
* **UI/UX:** Responsive design with CSS grid and styling

---

## Setup Instructions

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Arsalan-Azhar-AI/Tic-Tac-2-Game.git
   ```
2. **Open the game:**
   Open `index.html` in any modern browser (Chrome, Firefox, Edge, Safari).
3. **Play the game:**

   * Players click on empty squares to place their marks.
   * The game announces the winner or draw automatically.
   * Click the **Reset Game** button to restart.

---

## Usage Example

1. Player 1 clicks a square → "X" appears.
2. Player 2 clicks a square → "O" appears.
3. Continue until one player wins or the board is full.
4. Reset and start a new game anytime.
